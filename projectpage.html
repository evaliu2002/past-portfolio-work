CSE 332 (Data Structures and Parallelism Project 1

Partner Project

Used java to work with a WorkList ADT which is a generalization of StackS, QueueS, etc. The ADT contains items to be processed in certain orders and is defined as follows:
add(work) - Notifies the worklist that it must handle work.
peek() - Returns the next work to work on.
next() - Removes and returns the next work to work on.
hasWork() - Returns true if there's any work left and false otherwise.
We were assigned to write several implementations of the ADT: ArrayStack, ListFIFOQueue, and CircularArrayFIFOQueue.
